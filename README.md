# PRUEBA-INTRODUCCION

Este proyecto consiste en la creación de una tabla en una página web, con el objetivo de practicar y comprender mejor el uso de HTML y CSS.

## Tecnologías y herramientas utilizadas
- HTML
- CSS
- Live Server
- Diseño con paleta de colores personalizada
